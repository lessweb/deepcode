# Deepcode

Deepcode sidebar chat for OpenAI-compatible models.

## Usage

- Open the Deepcode view in the Activity Bar.
- Enter a prompt and click the send button.

## Configuration

Create `~/.deepcode/settings.json` with:

```json
{
  "env": {
    "MODEL": "deepseek-chat",
    "BASE_URL": "https://api.deepseek.com",
    "API_KEY": "sk-..."
  }
}
```
